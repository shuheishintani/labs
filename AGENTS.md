# AGENTS.md

このリポジトリは、学習・検証・プロトタイピングを目的とした小規模なプロジェクトを管理します。

## 方針

- 作業を始める前に、変更対象のディレクトリ（プロジェクト）を最初に確認し、以降はその範囲に限定する。
- 学習のため、まずはシンプルで読みやすい実装を保つ（過度な抽象化や先回りの最適化は避け、必要になってから拡張する）。
- 理解を助けるため、必要な箇所に限って日本語コメントを付ける。
  - 自明なコメントは不要。
  - 「何をしているか」よりも「なぜそうするか（WHY）」を優先する。
- エラーメッセージおよびテストケース名は英語で記述する。

## 作業時のチェックリスト（完了条件）

- [ ] 実装が動作すること
- [ ] テストが追加・更新され、すべて通ること
- [ ] README が更新され、使い方や意図が説明されていること
- [ ] 変更が必要最小限に抑えられていること
